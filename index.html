Project IJslander

An application developed for IJslander B.V. to simulate future playgrounds.

Requested by IJslander B.V. We created an application that allows the user to select a field and place playground equipment to their liking to give the user an idea of how it would look in the real world. With this application potential customers get a view of how their playground would look like before it's build.
